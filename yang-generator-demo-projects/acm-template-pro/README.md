# acm-template-pro-generator
> ACM 示例模版生成器
> 
> 作者： YANG
> 
> 基于 [yang-generator] 项目制作

可以通过命令行交互式输入的方式动态生成想要的代码

# 使用说明
执行项目跟目录下的脚本文件：
> generator <命令> <选项参数>

示例命令：
> generator generate -l -a -o

# 参数说明
| 参数 | 类型 | 默认值   | 缩写 |
| ---- | ---- |-------| ---- |
| loop | 布尔值 | false | -l |
| author | 字符串 | YANG  | -a |
| output |字符串 | null  | -o |
